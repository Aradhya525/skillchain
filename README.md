# SkillChain 🔗

**Opportunities defined by skills, not by money.**

SkillChain is a skill-barter platform built for the **Fastathon Hackathon** by **Team Hackattack**.

## SDG Focus
- **SDG 8** — Decent Work and Economic Growth
- **SDG 4** — Quality Education

## Problem Statement
Students and individuals from low-income backgrounds often have valuable skills but lack access to job opportunities, monetization platforms, and reliable networks to exchange their abilities. Most existing systems are money-driven, which excludes those without financial resources.

## Solution
A platform that enables users to exchange skills directly without using money. Users trade skills based on mutual needs, encouraging both learning and earning simultaneously.

## Core Features
- AI-based skill matching
- Smart skill recommendations
- Fair value exchange system
- User ratings and trust scores
- Real-time chat and session booking
- Progress tracking and gamification badges

## Tech Stack
- **Frontend**: React + Vite + TailwindCSS
- **Backend**: Node.js + Express 5
- **Database**: PostgreSQL + Drizzle ORM
- **API**: OpenAPI 3.1 with code generation
- **Monorepo**: pnpm workspaces + TypeScript
